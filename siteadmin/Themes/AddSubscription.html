Subscription Explaination
"Subscription Title" and "Subscription Description" - These two fields are both displayed to the end user when they are viewing the ViewSubscriptions page on the front end of the script. The title will also appear beside their username on items they are selling.

"Add to User Group" - This is where the main features of subscriptions come from. Create User Groups with the Category listing permissions, discounts, and tokens that you want members of this subscription plan to have, then use the above box to assign that group to this plan and the users will automatically be added and removed from that group depending on their subscription status.

"Duration" - How long (in days) a user will be subscribed to this plan. Can be 'unlimited' and the subscription will never end.

"Duration Price" - How much it costs to be subscribed for the duration listed above. Can be 0 (Zero), if it is zero, all the following payment options will be ignored.

"Recurring" - Creates an Automatically Recurring subscription plan so that users are automatically charged and renewed at the end of each 'Duration'. This option is only available through paypal. If you are going to use this method you should make sure that your users are aware that the subscription automatically charges their paypal account at the end of their subscription to renew the subscription otherwise you are likely to have some angry customers when they find out the hard way that they were billed again.

"Intro Duration and Price" - This can only be used by paypal recurring payment plans, it allows you to have a special introductory offer for your recurring subscriptions. For example if you have a 30 recurring subscription for $20 every duration, you could offer the first month (or what ever duration you chose) at only $10 (or whatever price you chose) then all the following durations would be normal, as listed above.

"Icon" - An Icon is a small image that will show up by the subscribed user's name on their listings and in their MyAccount area (also on the ViewSubscriptions page). This is especially useful if you have some kind of subscription that is used to "Verify" serious sellers, you can have an "I'm Verified" icon by their username on their listings so that buyers know they arern't just some one who is trying to rip them off, but they have been 'authenticated' some way by this site. Upload your icon image to your '/images' directory then type it's file name in the above box.

"Accept New Signups" - This will tell the system if a subscription plan should be active. If you are ending a plan offering while some users are still subscribed then it's a good idea to simply 'deactivate' it here instead of deleting it till all users' subscriptions have expired. Unchecking this box will make it so that this plan will not show up as an option on the ViewSubscription page and will not accept new subscribers. This can also be useful for special cases, such as a subscription plan intended for only a handful of users or in the 'verification' idea mentioned above, a subscription plan can be created, but never made public so you can manually add users to it under "Subscribed Users" here in siteadmin.
