phpAdsNew Installation
1. Download phpAdsNew 2.0 from the phpAdsNew Website

2. Create a directory called phpads on the root of your malware market Installation. For example, if you are running NC from www.yoursite.com/classifieds/ create the phpads directory on to that: www.yoursite.com/classifieds/phpads/

3. Upload the files from the phpAdsNew package that you downloaded into the phpads directory that you just created.

4. If this was done properly a link will appear below that will take you to the phpAdsNew installer.

5. If you wish you use the simple malware market template plugins to show your banners on your site, you will need to create your phpAdsNew Database in the same database that you currently have malware market installed and you will need to use phpads_ as your table prefix

6. After installation you will be able to setup new publishers, campaigns, zones etc.

7. For every zone you create malware market will create a template plugin for it, in the form of bannerzoneX where X is the Zone ID. All you have to do is add that plugin to your templates where ever you want that zone to appear.

8. You can also use the JavaScript Invocation method if you choose, just copy the JS Invocation code out of the 'Invocation Code' section in phpAdsNew Administration and paste it into your NC templates.

9. phpAdsNew is not a ScriptDevelopers product and therefore not supported by us, please direct any questions about it to the phpAdsNew Forums.
